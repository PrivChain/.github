# Support

## Primary Support Channel

- General support and partner inquiries: `hello@privchain.io`

## Before Opening an Issue

1. Check repository README and docs.
2. Check open and closed issues for duplicates.
3. Confirm this is not a security vulnerability.

## Issue Routing

- Bugs: use bug report template
- Feature requests: use feature request template
- Questions: use question template or Discussions (if enabled)
- Security vulnerabilities: email `hello@privchain.io` privately

## Stage Transparency

Project status and release-gate visibility are published at:
